# rn-invoke

Invoke any native code directly from Javascript in React Native

## Installation

```sh
npm install rn-invoke
```

## Usage

```js
import RnInvoke from "rn-invoke";

// ...

const result = await RnInvoke.multiply(3, 7);
```
